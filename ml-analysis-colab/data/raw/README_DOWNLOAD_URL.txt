# Añadir el dataset raw (heart.csv)

Para obtener el dataset de ejemplo puedes descargarlo desde:
https://raw.githubusercontent.com/ChiragSahni/Data-Science-Heart-Disease/master/heart.csv

Descarga el archivo y colócalo en esta carpeta: `data/raw/heart.csv`

Alternativamente, en Google Colab puedes leerlo directamente desde la URL con:
```python
url = "https://raw.githubusercontent.com/ChiragSahni/Data-Science-Heart-Disease/master/heart.csv"
df = pd.read_csv(url)
```
